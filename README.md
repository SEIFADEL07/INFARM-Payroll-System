# نظام إدارة وتدقيق الرواتب المالي (RTL Arabic)
## HR Salary Management & Auditing System

تطبيق ويب حديث ومتطور لإدارة كشوفات ومسيرات الرواتب للموظفين باللغة العربية (RTL)، يدعم قراءة ملفات Excel تلقائياً، وتدقيق العمليات الحسابية للرواتب وتوليد قسائم رواتب فردية بصيغة PDF وتصدير كشوفات معدلة إلى Excel، بالإضافة إلى لوحة تحكم إحصائية تفاعلية ومساعد مالي مدعوم بالذكاء الاصطناعي (يعمل دون اتصال بالإنترنت أو باستخدام مفتاح Gemini).

---

## الميزات الرئيسية (Core Features)

1. **واجهة عربية متكاملة (RTL):** تصميم واجهة مستخدم مظلمة (Dark Mode) حديثة وزجاجية (Glassmorphic) تدعم القراءة من اليمين لليسار بشكل طبيعي ومريح للعين مع إمكانية التبديل للوضع المضيء.
2. **رفع وقراءة ملفات Excel تلقائياً:** تحليل أوراق عمل متعددة دفعة واحدة ورسم خريطة الأعمدة تلقائياً بناءً على المرادفات اللغوية للكلمات (مثل: الاسم، القسم، الأساسي، البدلات).
3. **تدقيق وحساب صافي الرواتب:** احتساب تلقائي لصافي الرواتب ومقارنتها بالقيم المسجلة بالملف، مع التحقق من الأخطاء الحسابية (تجاوز الخصومات للراتب، الراتب الأساسي الصفري، الفروقات الحسابية).
4. **تصدير التقارير المتقدمة:**
   - **قسيمة راتب الموظف (PDF):** توليد قسيمة راتب رسمية ومنسقة بالكامل للموظف بضغطة زر.
   - **كشف المسير الإجمالي (PDF):** طباعة وحفظ التقرير الإجمالي للمسير الحالي مع الإجماليات والتوقيعات الإدارية.
   - **تصدير ورقة Excel:** توليد ملف Excel جديد تماماً ومنسق باللغة العربية يتضمن كافة الحسابات والتدقيقات المحدثة.
5. **مساعد مالي ذكي (AI Assistant):** 
   - **الوضع المحلي (دون إنترنت):** محرك لغوي محلي متطور يجيب على الأسئلة الشائعة للرواتب (مثل: ما إجمالي الرواتب؟ من أعلى موظف راتباً؟ هل توجد أخطاء بالمسير؟).
   - **الوضع المتقدم (Gemini API):** ربط اختياري مع Google Gemini لإجراء تحليلات مالية متطورة وكتابة مقارنات معقدة.
6. **حفظ وتخزين البيانات:** الاحتفاظ بكافة الملفات المرفوعة، الإعدادات، والمراسلات حتى بعد إعادة تحميل الصفحة عبر التخزين المحلي الآمن بالمتصفح (`localStorage`).

## دليل تشغيل التطبيق للمبتدئين (Setup Instructions for Beginners)

لأن التطبيق مصمم ليعمل بالكامل على جانب العميل (Client-Side)، فلا توجد قواعد بيانات معقدة أو عمليات تثبيت مطولة. للتشغيل، اتبع الخطوات البسيطة التالية:

### الطريقة الأولى: التشغيل التلقائي عبر Python (موصى بها)

يأتي التطبيق مزوداً بخادم محلي صغير مكتوب بلغة بايثون لتجنب قيود أمن المتصفح للملفات المحلية.

1. تأكد من تثبيت لغة **Python** على جهازك (تأتي مثبتة افتراضياً في معظم الأجهزة، أو يمكنك تحميلها من موقع [python.org](https://www.python.org)).
2. افتح موجه الأوامر (Terminal / Command Prompt) في مجلد المشروع.
3. قم بتشغيل الأمر التالي:
   ```bash
   python server.py
   ```
4. سيقوم الخادم بالعمل فوراً على المنفذ `8000` وسيقوم **بفتح المتصفح تلقائياً** على الرابط: `http://localhost:8000`.

### الطريقة الثانية: فتح الملف مباشرة
إذا لم يكن لديك بايثون، يمكنك ببساطة النقر المزدوج على ملف `index.html` لفتحه في أي متصفح ويب (Chrome, Edge, Safari)، ولكن يفضل استخدام الطريقة الأولى لضمان التوافقية الكاملة للملفات وعمليات التنزيل.

## كيف تجرب التطبيق؟ (How to Test)

1. عند فتح التطبيق لأول مرة، اضغط على زر **"تحميل ملف تجريبي (Sample)"** في صفحة الرفع.
2. سيقوم النظام بتحميل ملف Excel مهيأ مسبقاً يحتوي على أسماء موظفين عشوائية، أقسامهم، رواتبهم الأساسية وبدلاتهم.
3. قم بسحب الملف التجريبي الذي تم تحميله وإفلاته في منطقة الرفع المخصصة.
4. ستفتح لك لوحة التحكم فوراً لتعرض الإحصائيات، الرسوم البيانية، جدول الرواتب والمساعد الذكي!

## هيكلية مجلدات المشروع (Folder Structure)

```text
HR-Salary-Management/
│
├── index.html            # الواجهة الرسومية وهيكل التطبيق الرئيسي
├── README.md             # دليل التشغيل والتعليمات (هذا الملف)
├── server.py             # خادم بايثون للتطوير المحلي السريع
│
├── css/
│   ├── style.css         # ملف التنسيق الزجاجي الداكن وتأثيرات الحركة
│   └── responsive.css    # ملاءمة التطبيق للهواتف والأجهزة اللوحية
│
└── js/
    ├── app.js            # منسق الأحداث والعمليات ومزامن التخزين المحلي
    ├── parser.js         # محلل ملفات Excel ومطابق الأعمدة الذكي
    ├── calculator.js     # محرك العمليات الحسابية وتدقيق الأخطاء الحسابية
    ├── exporter.js       # مصمم ومصدر تقارير PDF وجداول Excel المحدثة
    ├── charts-custom.js  # مصمم الرسوم البيانية للرواتب والأقسام
    └── ai.js             # المساعد المالي الذكي (المحلي وعبر Gemini API)
```

## English Quick Start

1. Ensure **Python** is installed.
2. Open terminal in this folder and run:
   ```bash
   python server.py
   ```
3. The browser will automatically launch at `http://localhost:8000`.
4. Click **"Download Sample"** to get a dummy payroll Excel sheet, then upload it to experience the full features of the interactive dashboard, automated validations, PDF salary slip generator, and AI chatbot.
